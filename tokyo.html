Scrollersite

Background

The purpose of this project was to learn how to use scrolling related properties. We had to build a single page website that utilises as many scrolling features (parallax effect, overflow etc.) as possible. The page should contain several visually different sections, a fixed element and media that reacts to scrolling.

Idea

The content was up to us to I decided to make a page for people who would like to discover Tokyo.

Process

We did scrolling related exercises in class. I also looked up webpages that use scrolling effects to get inspiration. I tested the effects that I decided to use seperatly and put them all together in the end. In order to come up with the section content, I created a small persona.
